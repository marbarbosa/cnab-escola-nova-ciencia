# CNAB 240 - ☀️Escola Nova Ciência

Esse projeto é para gerar o arquivo CNAB 240, referente aos boletos bancários das mensalidades dos alunos da Fundação Escola Nova Ciência.

## 🚀 Começando

Os dados para a extração do arquivo CNAB 240 está em uma planilha Excel.
Essa planilha deve ser informada no arquivo app.config.

### 📋 Pré-requisitos

Layout da planilha Excel

| Nro. | Nome | Pai/Mãe | Dia Vencto | Valor | Cantina | Conteúdo Digital | Mat. Diático Inglês (1/3) | Total | e-mail 1 | e-mail 2 | # Boleto anterior | # Boleto atual | Observações |
| ---- | ---- | ------- | ---------- | ----- | ------- | ---------------- | ------------------------- | ----- | -------- | -------- | ----------------- | -------------- | ----------- |
| 1 | Joaquim | Andrea | 5 | 1,001.99 | 57.80 | 67.09 | 71.09 | 1,201.99 | email1@email.com | email2@email.com | 8506 | 8538 | Desconto 10% |
